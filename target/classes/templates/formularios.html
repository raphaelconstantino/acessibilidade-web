Associar <label> com attributo ID

Recomenda-se fazer a relação adequada dos atributos IDs dos formulários com seus respectivos rótulos,utilizando o elemento <label>.

Usar alt para botões do tipo image

No caso de botões com <input type="img">, utilize o atributo alt para indicar a ação a ser feita; por exemplo, alt=”enviar formulário”.
Essa técnica garante que tecnologias assistivas possam relacionar adequadamente os respectivos textos com seu campo de formulário

Agrupar blocos de formulário confome dados utilizando LEGEND e FIELDSET

É recomendável agrupar blocos de formulário conforme o tipo de dado solicitado.
Para evitar que um campo apareça duas vezes sem sua devida marcação de grupo, utilize o elemento fieldset para agrupar os blocos e o elemento legend para declarar o título adequado daquele grupo.

Usar aria-required="true" para campos obrigatórios

Muitas páginas têm problemas com acessibilidade por se preocuparem apenas em expressar visualmente as restrições.
Por exemplo, um formulário que indica os campos obrigatórios apenas através da mudança de cor do input. Um usuário que não pode ver, não saberá quais campos são de preenchimento obrigatório.
A solução é justamente usar aria-required="true"

Usar area-label para campos input

Usamos novamente no botão um aria-label para sobrescrever a descrição utilizada pelo leitor de tela para o componente.
