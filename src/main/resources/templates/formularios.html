Associar <label> com attributo ID

Recomenda-se fazer a relação adequada dos atributos IDs dos formulários com seus respectivos rótulos,utilizando o elemento <label>.

Usar alt para botões do tipo image

No caso de botões com <input type="img">, utilize o atributo alt para indicar a ação a ser feita; por exemplo, alt=”enviar formulário”. Essa técnica garante que tecnologias assistivas possam relacionar adequadamente os respectivos textos com seu campo de formulário
