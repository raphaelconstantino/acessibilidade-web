Equivalente Textual

Todas as informações de uma página acessível devem ser apresentadas em texto. Isso significa que, se for usada alguma outra mídia, como imagens e sons, as informações que elas contêm devem ser repetidas numa descrição textual. Essa descrição deve ser "equivalente", isto é, deve transmitir as mesmas informações que os elementos disponibilizados.

Atributo ALT

Use o atributo ALT para objetos não textuais simples. Deve-se especificar a descrição textual utilizando o atributo HTML "Alt" (alternativo). O texto do atributo ALT deve ser escrito cuidadosamente para que possa fornecer informações equivalentes. Para imagens meramente decorativas que não contêm informações, tais como espaçadores, linhas e enchimentos, usa-se o texto Alt vazio

Fornecimento de uma longa descrição quando o texto Alt não for adequado.

O atributo ALT não é útil para explicações complexas demais para serem descritas em poucas palavras. Em casos que a descrição é muito longa deve-se utilizar o atributo HTML "LONGDESC" ou um "D" link. Essas duas técnicas permitem aos usuários acessarem um texto separado que descreve o contexto. Longdesc, embora seja parte da especificação do HTML, não é suportado por grande parte dos navegadores. A técnica alternativa "D" link envolve a inserção, além da imagem, de um link apresentado como a letra maiúscula "D". O alvo desse link deve ser uma página web separada com uma longa descrição detalhada para pessoas com deficiência visual. Quando o usuário tiver lido esse texto, ele pode retornar à página com a imagem. "Longdesc" significa "longa descrição".
