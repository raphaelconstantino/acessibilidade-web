Implementando uma Modal

Utilizando as Arias podemos deixar as modals mais sensíveis aos leitores de tela. O atributo role="dialog" informará ao leitor que ele está dentro de uma caixa de diálogo. Além disso, o aria-labelledby possui valor do ID do elemento que descreve a caixa de diálogo, que pode ser o título, assim o leitor informa o usuário o nome desse modal.
Outro atributo muito utilizado é aria-label aplicado no botão de fechar. O botão foi construído com apenas um "x”, que é adequado para usuários visuais saberem que serve para fechar. Mas,para leitores de tela, usamos uma frase melhor, com aria-label="Fechar". O leitor vai interpretar esse valor na hora de ler para o usuário.

<div id="myModal" role="dialog" aria-labelledby="titulo">
  <div class="modal-dialog modal-sm">

	<div class="modal-content">
	  <div class="modal-header">
		<button type="button" class="close" data-dismiss="modal" aria-label="Fechar Modal" >×</button>
		<h4 class="modal-title">Título da Modal</h4>
	  </div>
	  <div class="modal-body">
		<p>Conteúdo do modal de exemplo</p>
	  </div>
	  <div class="modal-footer">
		<button type="button" data-dismiss="modal" aria-label="Fechar Modal">Fechar</button>
	  </div>
	</div>

  </div>
</div>
